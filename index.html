Overview

gh-pages Branch는 Web Page 제공을 위해 Git Hub Repository에서 특수하게 제공하는 Branch이다.

http://[Account ID].github.io의 도메인으로 호스팅되며, index.html 페이지를 Default Page로 간주한다.

이 페이지는 index.html 파일이이며, 아래는 마크다운을 이용하여 만든 슬라이드이다.

실제 Jekyll도 이 브랜치를 이용하여 문서화를 한다.
